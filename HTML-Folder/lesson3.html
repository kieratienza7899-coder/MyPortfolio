Computing System

Definition

A computing system is a set of components that work together to accept input, process data, store information, and produce output, using hardware, software, and human interaction.

Components

Input Unit - Accepts data from users (e.g., keyboard, mouse, scanner).

Processing Unit (CPU)- Performs calculations and controls operations.

  • ALU: Arithmetic and logical operations
  • Control Unit: Directs data flow

    • Storage Unit - Stores data temporarily (RAM) or permanently (HDD/SSD).

    Output Unit - Presents processed information (monitor, printer, speakers).

    Software

  • System software: OS, drivers
  • Application software: Programs for specific tasks (Word, Excel)

    • Data Flow

    Input → Processing → Storage → Output

    Example: Typing a document → CPU formats → File saved → Displayed on screen

    Types of Computing Systems

  • Personal Computers, Mobile devices, Servers, Embedded systems, Supercomputers

    • Summary

    A computing system processes information using its core components: Input, Processing, Storage, Output, guided by software.
