Digital Logic System

Definition

A digital logic system processes binary signals (0 = LOW, 1 = HIGH) to perform operations. It is the foundation of all digital devices and computers.

Logic Gates

Gate, Symbols, Function, Boolean.

AND, &, 1 if both inputs 1, A·B

OR, ≥1, 1 if any input 1, A + B

NOT, ¬, Inverts input, ¬A

NAND, ¬&, NOT AND, ¬(A·B)

NOR, ¬≥1, NOT OR, ¬(A + B)

XOR, ⊕, 1 if inputs differ, A⊕B

XNOR, ≡, 1 if inputs same, A≡B

Boolean Algebra

  • Variables: 0 or 1
  • Simplifies circuits
  • Key laws: Commutative, Associative, Distributive, De Morgan

    • Circuit Types

  • Combinational: Output depends only on current inputs (e.g., adders, multiplexers)
  • Sequential: Output depends on current inputs and past states (e.g., flip-flops, counters)

    • Applications

  • Computers & processors
  • Digital watches & calculators
  • Memory devices & communication systems

    • Summary

  • Digital logic uses binary signals.
  • Logic gates process inputs to produce outputs.
  • Boolean algebra helps simplify logic circuits.
  • Circuits are either combinational or sequential.
