Number Systems

Definition

A number system is a way of representing numbers using a consistent set of symbols and rules.

Common Number Systems

Number System Base Symbols Used Example

Binary 2 0, 1 1010₂ = 10₁₀

Octal 8 0-7 12₈ = 10₁₀

Decimal 10 0-9 10₁₀ = 10

Hexadecimal 16 0-9, A-F A₁₆ = 10₁₀

Number System Conversions

Binary ↔ Decimal

  • Binary → Decimal: Multiply each bit by 2^position and sum.

    • Example: 1101₂ = 1x2³ + 1x2² + 0x2¹ + 1x2⁰ = 13₁₀

  • Decimal → Binary: Divide the number by 2 repeatedly and write remainders in reverse.

    • Example: 13₁₀ → 1101₂

    Decimal ↔ Octal / Hexadecimal

  • Decimal → Octal/Hex: Divide by 8 or 16, write remainders in reverse.

    • Example: 31₁₀ → Hex: 31 ÷16 = 1 remainder 15 → 1F₁₆

  • Octal/Hex → Decimal: Multiply digits by powers of base.

    • Example: 2A₁₆ → 2x16¹ + 10x16⁰ = 42₁₀

    Binary ↔ Octal / Hex

  • Binary → Octal: Group binary digits in 3s from the right.

    • Example: 110101₂ → 110 101 → 6 5 → 65₈

  • Binary → Hex: Group binary digits in 4s.

    • Binary → Hex: Group binary digits in 4s.

  • Octal/Hex → Binary: Convert each digit to binary.

    • Example: F₁₆ → 1111₂

    Binary Arithmetic (Basic)

  • Addition: 0+0=0, 0+1=1, 1+1=10 (carry 1)
  • Subtraction: 0-0=0, 1-0=1, 1-1=0, 0-1=1 (borrow 1)
  • Multiplication: Like decimal but 0x0=0, 0x1=0, 1x1=1
  • Division: Repeated subtraction in binary

    • Example: 101₁₂ + 11₂ = 1000₂

    Applications

  • Binary: CPU operations, memory addressing
  • Octal/Hex: Programming, debugging, network addresses
  • Decimal: Human-readable data

    • Summary

  • Number systems use a base and symbols.
  • Binary, Octal, Decimal, Hexadecimal are commonly used in computing.
  • Conversions between systems are essential for computer science and digital electronics.
  • Binary arithmetic forms the basis of all digital computations.
